# Lexemo

A modern, web-based language learning game that uses the **FSRS (Free Spaced Repetition Scheduler)** algorithm to optimize vocabulary retention.

- **FSRS Integration**: Scientific scheduling for long-term memory.
- **Free Mode**: Practice specific level ranges at your own pace.
- **Pronunciation Mode**: Use your microphone to practice speaking (Web Speech API).
- **Custom Decks**: Import/Export your own vocabulary via JSON.
- **Premium UI**: Sleek dark mode, glassmorphism, and smooth animations.
